ЛАБОРАТОРНА РОБОТА

Тема: Об’єктно-орієнтоване програмування в Python (класи та їх основні конструкції)
Мета роботи
Ознайомитися з принципами об’єктно-орієнтованого програмування в мові Python; навчитися створювати класи та об’єкти, працювати з методами, властивостями, статичними та класовими методами, а також використовувати магічні методи.

Виконання роботи
Завдання 1: Створення базового класу
Було створено базовий клас MyAnimals без атрибутів і методів. Створено об’єкт цього класу та перевірено його тип.
Результат виконання:
<class '__main__.MyAnimals'>

Завдання 2: Порівняння простих даних з об’єктами
Було створено списки з даними про тварин, а також клас MyAnimals з конструктором.
На основі класу створено об’єкти для таких тварин:
собака Берта (3 роки),
хомʼяк Джексон (2 роки),
кіт Чарлі (6 років).
Результат виконання:
Берта Джексон Чарлі

Завдання 3: Методи класу
У класі реалізовано методи:
speak() — повертає звук тваринки;
add_one_year() — збільшує вік на один рік.
Методи були застосовані до всіх створених об’єктів.
Результат виконання:
Берта говорить Гавв  
Джексон не розмовляє  
Чарлі говорить Мяууу

Завдання 4: Змінні класу
Було використано змінні класу для підрахунку загальної кількості тварин та збереження їх типів.
Результат виконання:
3 {'Собака', 'Хомʼяк', 'Кіт'}

Завдання 5: Статичні методи
У класі реалізовано статичний метод call_pet(), який не залежить від конкретного об’єкта. Метод було викликано як через об’єкти, так і безпосередньо через клас.
Результат виконання:
Тваринка біжить до нас

Завдання 6: Властивості (Properties)
Було реалізовано властивості:
live_untill — рік, до якого тваринка може прожити;
age_in_months — вік у місяцях.
Також використано метод додавання одного року.
Результат виконання:
2039  
72

Завдання 7: Методи класу та робота з CSV
Створено метод класу from_csv() для створення об’єктів із рядка CSV та метод перевірки правильності написання імені з великої літери.
Результат виконання:
Ім'я 'Берта' задано правильно  
Ім'я 'rex' має починатися з великої букви

Завдання 8: Магічні та арифметичні методи
У класі реалізовано магічні методи:
__str__, __repr__ — для виведення об’єкта;
__sub__ — взаємодія між об’єктами та числами;
__gt__ — порівняння тварин за віком;
__len__ — визначення довжини імені.
Результат виконання:
Берта покусала Чарлі  
Берта постаріла на 1 років, тепер йому 4  
Берта не старша за Чарлі  
5
